The Journey to MSc in Cyber Security - University of Essex

Module1-Launching into cyber security

In summary, cyber security is becoming a persistent need in every aspect of life. economic activities and national security depend on stability, safety, and resilient cyberspace. A network brings communications and transport, power to homes, runs the economy, and provides governments with various services. However, the same cyber networks intrude and attack our privacy, economy, and social life in a harmful way. Madhu T., 2017). Therefore, for security to be robust at any organization, be it a network, software, mobile application, or an IoT device or operating system. A suitable design model should be used to implement an efficient product with the initial essential security functions. for software, security should be part of the Software Development Life Cycle (SDLC) allowing the detection of defects and lowering the cost of fixing errors before the release (James R. 2021). After product release, a plan to improve the security posture should be put as part of the security audit and assessment process (Kris H. 2023). Furthermore, a key element in security is following a framework to reduce security risk, the organization’s requirements are essential in selecting the suitable framework, HIPAA for health sectors in the US or the Data Protection Act 2018 for the UK. Setting policies and rules and following GDPR is as can be described as “road to compliance” (Kris H., 2023). Plethora of examples of how data breaches can be devastating to society, like the Shell Group data breach, the impact on user information being leaked is a failure in protecting the triad of data CIA, which is an important concept in the information security industry and is used in ISO 27001, a global standard for managing information security. The GDPR also mentioned in Article 32, that it requires organizations to use appropriate measures to protect the confidentiality, integrity, availability, and resilience of their information processing systems and services(Luke I., 2023). Reference: James, R. Brook, S. (2021). Building in Security at Agile Speed:Auerbach Publications. Madhu, T. (2017). Security Against Cyber-Crime:Horizon Books. Kris, H. (2023). Mastering Cyber Security: Cybellium Ltd. Kris, H. (2023). Mastering GDPR in the Enterprise : Cybellium Ltd. Luke, I. (2023). What is the CIA Triad and why is it important Online at: https://www.itgovernance.co.uk/blog/what-is-the-cia-triad-and-why-is-it-important [Accessed on: 17 Feb. 2024].
